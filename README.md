# Playwright Automation Framework with TypeScript

This project is an automated testing framework built with **Playwright** and **TypeScript**.

The primary goal of this project is learning and improving automation testing skills while implementing industry best practices such as the **Page Object Model (POM)** design pattern.

Created by **Alain Soto** as part of a continuous learning journey in test automation using Playwright.

## Technologies Used

This framework was developed using the following technologies and tools:

### Test Automation

- **Playwright** - End-to-end testing framework for modern web applications.
- **TypeScript** - Strongly typed programming language built on JavaScript.
- **Node.js** - JavaScript runtime environment.

### Framework Design

- **Page Object Model (POM)** - Design pattern for improving test maintainability and reusability.

### Version Control

- **Git** - Distributed version control system.
- **GitHub** - Source code hosting and repository management platform.

### Continuous Integration & Delivery

- **GitHub Actions** - Automation workflows for test execution and CI/CD pipelines.
- **Jenkins** - Continuous Integration and Continuous Delivery (CI/CD) automation server.

### Scripting & Automation

- **PowerShell** - Task automation and environment management scripting language.

### AI-Powered Development

- **GitHub Copilot** - AI-powered coding assistant used to accelerate development and improve productivity.
- **Claude AI** - AI assistant used to support framework design, code optimization, documentation, and learning activities.
